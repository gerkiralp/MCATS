# Machine Learning Cryptocurrency Automated Trading System (MCATS)

MCATS is an automated trading system for cryptocurrencies, leveraging machine learning for market prediction and automated trading execution.

## Project Structure

- `configuration/`: Configuration settings for the trading system
- `data_layer/`: Data acquisition and processing pipeline
- `feature_engineering/`: Feature creation and preprocessing
- `market_execution/`: Market order execution logic
- `order_risk_management/`: Risk management and position sizing
- `trading_agent_layer/`: Trading algorithms and ML models
- `training_pipeline/`: Model training and evaluation
- `user_interface/`: User interface components
- `notebooks/`: Jupyter notebooks for analysis and experimentation
- `tests/`: Unit and integration tests

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/gerkiralp/MCATS.git
   cd MCATS
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the main application:
```
python mcats_main.py
```
